# Tab Title Picker - Chrome Extension

A simple Chrome extension that displays the current tab's title when you click a button.

## Features

- Clean, minimal interface
- One-click tab title display
- Built with Manifest V3
- No external dependencies

## Installation

1. Open Chrome and go to `chrome://extensions/`
2. Enable **Developer mode** (top-right toggle)
3. Click **Load unpacked** and select this folder
4. Click the extension icon and press the button to see the current tab's title

## Files

- `manifest.json` - Extension configuration
- `popup.html` - Popup interface
- `popup.css` - Minimal styling
- `popup.js` - Tab title retrieval logic

## Technologies

- HTML5, CSS3, JavaScript (ES6)
- Chrome Extension API (Manifest V3)
